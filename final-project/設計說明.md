## 一、主題定位（我為什麼做這個主題）

三個主題裡，我選了「個人作品集」。原因很單純：我手上其實累積了不少東西——邊緣運算的硬體專案、PM2.5 的深度學習模型、一堆競賽獎項——可是這些一直散落在各個資料夾和檔案裡，沒有一個地方能「一次完整呈現我是誰、我會什麼」。我希望做一個網站，是那種「丟一個網址給教授、或之後找實習時丟給對方，他點開就能大致認識我」的東西。

- **目的**：用一個能被別人打開的網址，完整呈現我在 AI、邊緣運算、硬體實作上的能力與成果。
- **對象**：課程教授與評審、未來的實習／升學審查者，以及對這個領域有興趣的同行。
- **調性**：因為我做的東西本來就跟晶片、資料、AI 有關，所以我想走「科技、俐落、有點工程師味道」的風格，而不是花俏可愛那種。

## 二、設計概念與風格

整個網站我想圍繞一個畫面感：**「程式碼與資料在流動的科技空間」**。從這個核心去延伸所有的視覺決定。

**為什麼用深色？** 一方面深色介面在工程／AI 圈本來就很常見，看起來比較專業；另一方面深底才能把我選的霓虹點綴色襯托出來，亮起來很有記憶點。後來我也額外做了一套「米黃暖色」的淺色版本，讓不習慣深色的人可以切換。

幾個我特別花心思、也最能代表這個概念的設計：

- **開場解碼動畫**：網站一打開，會先跑一串隨機亂碼快速跳動，再逐字「解碼」成我的代號 `lizcyy`，最後黑幕往上滑開、內容浮現。我希望第一眼就把科技感講清楚。
- **數字雨背景**：四個區塊後面都有由上往下流動的 0 / 1 綠色數字，像資料流一樣。一開始我把它做得太亮、蓋住文字，後來我加了一層半透明遮罩壓暗，這樣既保留氛圍、文字又讀得清楚。
- **毛玻璃質感**：導覽列、技能卡片、競賽列表都用了半透明加背景模糊，讓後面的數字雨透出一層朦朧的光，做出懸浮的層次感。
- **雙主題、雙語**：左上角可以切深色／淺色，右上角可以切中文／English，而且會記住我上次的選擇。

## 三、設計系統（字體、色彩、間距）

我盡量讓整個網站「成一套系統」，而不是每個地方各自為政。

**字體**：英數用 `Inter`、中文用 `Noto Sans TC`（都來自 Google Fonts），這兩套都是為螢幕閱讀設計的字體，比系統預設字看起來精緻很多。再搭配抗鋸齒和稍微放大的行高，讀起來比較舒服。

**色彩**：我把所有顏色集中寫成 CSS 變數，這樣改一個地方全站連動，深淺色切換也是靠這套機制。

- 深色版：背景深藍黑 `#0d1117`、主色霓虹藍 `#00e5ff`、輔色紫 `#7b5cff`、數字雨螢光綠 `#00ff66`、得獎標籤紅 `#e23636`。
- 淺色版（米黃）：背景 `#f5efe0`、文字暖棕 `#3a3322`、主色 `#0088aa`。

**間距**：區塊之間留比較大的上下空白、標題下方也留足夠距離，讓畫面有「呼吸感」，不會擠成一團。圓角的部分，卡片用 12–14px，按鈕和標籤則用膠囊形。

## 四、Block / Flex / Grid 使用說明

這是這學期的重點，我特別把三種版面思維都用上，而且是「用在它最適合的地方」。

**▸ Block（區塊流）**
精選實作專案那一區，三個專案是用 `<article>` 做的區塊元素，靠瀏覽器預設的 Block 流由上往下自然堆疊，再用 `margin` 拉開彼此的間距。競賽榮譽區的小標題和清單也是這樣一塊一塊往下排。

**▸ Flexbox（彈性盒）**
這是我用最多的：
- 首頁的照片和自我介紹（電腦版左右並排、手機版上下排）。
- 自我介紹下方那排聯絡按鈕。
- 專案區的圖文並排，而且我用 `row-reverse` 做出「1、3 左圖右文，2 左文右圖」的交錯排版，讓畫面不會死板。
- 競賽榮譽改成單欄垂直列表，每一列「年份 + 名稱 + 標籤」也是用 Flex 對齊。
- 導覽列和頁尾的按鈕排列。

**▸ CSS Grid（網格）**
專業技術與能力那三張技能卡片，是用 Grid 排的，因為網格最適合做這種等寬卡片牆：電腦版三欄、平板兩欄、手機一欄。

## 五、響應式設計（RWD）

我希望網站在手機和電腦上都好看，所以設了幾個斷點去調整版面：

- 電腦（大於 1024px）：技能三欄、導覽列完整顯示。
- 平板（768–1024px）：技能改兩欄。
- 手機（768px 以下）：首頁圖文改成上下排並置中、專案圖在上文在下、聯絡按鈕置中。
- 很窄的手機（600px 以下）：把導覽列藏起來，避免跟角落的功能鈕擠在一起。

我有實際在手機寬度（約 375px）和電腦寬度（約 1200px）測試過，兩邊都能正常瀏覽。

## 六、互動與動畫

除了靜態版面，我加了一些小互動讓網站「活起來」：

- 開場的解碼動畫和黑幕揭幕轉場。
- 往下捲動時，各區塊的內容會淡入浮現（用 IntersectionObserver 偵測進入畫面才觸發），技能卡片還會稍微錯開時間登場。
- 滑鼠移到卡片、按鈕、圖片上時，會平滑地浮起並帶一層柔和光暈。
- 導覽列會跟著捲動高亮目前所在的區塊；捲到一半會出現「回到頂端」按鈕。
- 深淺色、中英文切換，而且都會記住我的選擇。

另外我也有考慮到，如果使用者在系統裡設定「減少動態效果」，網站會自動關掉這些動畫（無障礙考量）。
